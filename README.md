# C# con .Net Core 2.1

Console app with C# & .NET.

## Installation

Use the SDK from .NET [dotnet](https://dotnet.microsoft.com/en-us/download).

```bash
dotnet --version
```

## Usage

```csharp
# execute
dotnet run

# compile
dotnet build

# clean
dotnet clean

# help
dotnet --help

# new
dotnet new console
```
